# Project-Onion

## Features
* Create Post -> Photo/Blog
* Login/Register
* View Post of authors.

## Installation
**Requirement**<br>
1. Node<br>
2. NPM<br>

**Run**<br>

`
git clone https://github.com/abughalib/Project-Onion`<br>
`cd Project-Onion`<br>
`cd server`<br>
`npm install`<br>
`npm start
`<br>
